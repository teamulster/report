\subsection{Architecture}

The main goal of the software architecture of soHappy is providing flexability
in switching out various components.

Adding an Face- or Smile-Detector based on the technology of liking requires an
developer to implement the intended interface. The implementation then can be 
selected by modifing the configuration file of the app. A Factory Method will
create the Face- or Smile-Detector based on the configuration file.
